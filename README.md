# FuelEU Maritime Compliance Platform

## Overview
A minimal, deeply structured implementation of the Fuel EU Maritime compliance module exploring Routes, Compare, Banking, and Pooling.

## Architecture
- **Frontend**: React + TypeScript + TailwindCSS
- **Backend**: Node.js + TypeScript + PostgreSQL (via Prisma)
- **Design Pattern**: Hexagonal (Ports & Adapters / Clean Architecture) separating Core Logic from UI and Integration endpoints.
